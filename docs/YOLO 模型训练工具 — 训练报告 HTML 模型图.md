## 四、设计要点

| 要点 | 说明 |
|:---|:---|
| **图表来源** | Loss/mAP/P-R/F1/混淆矩阵等图直接使用 Ultralytics 训练输出的 PNG 文件，不重新绘制（避免不一致） |
| **逐类别指标** | 从 `results.csv` 中提取每个类别的最佳 AP，归并成表格 |
| **自动提示** | 混淆矩阵热力图中标注异常模式；类别不均衡时警告；弱势类别建议 |
| **筛选预测样例** | 默认展示随机抽样，支持"只看错误"——这是用户最关心的 |
| **中英双语** | 报告顶部提供语言切换，面向国际用户 |
| **打印友好** | CSS `@media print` 适配，隐藏按钮，图表分页不截断 |
| **自包含** | 图表以 Base64 内嵌 HTML，不依赖外部图片文件——报告可以离线发给任何人 |
