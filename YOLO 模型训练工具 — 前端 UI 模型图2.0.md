# YOLO 模型训练工具 — 前端 UI 模型图 2.0

---

## 图例说明

```
[ 文字 ]       = 普通文本标签 / 按钮文字
[ __________ ] = 文本输入框
[ ▼        ]   = 下拉选择框
[━━━●━━━]     = 滑块（带数值显示）
[ 📁 ]         = 文件夹/文件浏览按钮
[ 🔄 ]         = 刷新按钮
[ ✅ ]         = 校验通过/完成状态
[ ❌ ]         = 校验失败/错误状态
[ 🚀 ]         = 主要操作按钮（开始训练）
[ ⏸ ]          = 暂停操作按钮
[ ⏹ ]          = 停止操作按钮
[ 📷 ]         = 预览按钮
[ ⓘ ]          = 信息提示图标（hover 显示 Tooltip）
[ 📄 ]         = 查看报告
[ 📥 ]         = 下载
[ 🗑 ]         = 删除
[ 🔍 ]         = 搜索/推理测试
*               = 必填项
```

## 六、页面一：新建训练 — 训练中退出确认弹窗

弹窗内容根据训练进度动态变化：

**情况一：已产出模型文件时（训练至少完成 1 轮且保存了 checkpoint）**

```
┌────────────── 确认退出 ──────────────────────────────────────┐
│                                                               │
│  ⚠️  训练尚未完成，退出将中断当前训练。                       │
│                                                               │
│  已产出的模型文件（如 best.pt）会自动保留，                   │
│  之后可通过「训练历史」中的「断点续训」功能恢复使用。         │
│                                                               │
│  当前进度: 65/100 轮 (已用 18 分钟)                           │
│                                                               │
│            [ 留在当前页面 ]     [ 确认退出 ]                  │
│                                                               │
└───────────────────────────────────────────────────────────────┘
```

**情况二：尚未产出模型文件时（训练刚开始，无任何 checkpoint）**

```
┌────────────── 确认退出 ──────────────────────────────────────┐
│                                                               │
│  ⚠️  训练尚未完成，退出将中断当前训练。                       │
│                                                               │
│  当前尚未产出任何模型文件，退出后训练进度将丢失，             │
│  可从「训练历史」中重新开始该任务。                           │
│                                                               │
│  当前进度: 1/100 轮 (已用 30 秒)                              │
│                                                               │
│            [ 留在当前页面 ]     [ 确认退出 ]                  │
│                                                               │
└───────────────────────────────────────────────────────────────┘
```

- 确认退出用红色/警告色，留在当前页面用默认主色

## 设计原则总结

1. **三步即可开始**：上传数据集 → 上传模型 → 点击开始训练
2. **默认即最佳**：所有参数有合理默认值，小白用户无需理解即可使用
3. **透明可感**：训练状态全程可见（状态灯 + 进度条 + 日志），操作后果提前告知
4. **信息完整**：不因小白而省略数据——用户不懂可以截图问 AI
5. **消除脆弱感**：锁定 + 确认 + 可恢复，用户知道"碰哪里都不会出事"
6. **暗色主题**：默认深色界面，减少长时间注视的视觉疲劳
